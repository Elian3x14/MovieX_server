Introduction

Welcome to the MovieX documentation! This guide will help you get started with using MovieX, a powerful movie database application that allows you to search, filter, and manage your movie collection with ease.

Getting Started

To begin using MovieX, follow these steps:

  1. Install the application from our official website.
  2. Create an account to access all features.
  3. Import your existing movie collection or start adding new movies.

API Reference

MovieX provides a robust API for developers. Below are some of the key endpoints:

  • GET /movies: Retrieve a list of all movies in the database.
  • POST /movies: Add a new movie to the database.
  • GET /movies/{id}: Retrieve details of a specific movie by its ID.
  • PUT /movies/{id}: Update information for a specific movie.
  • DELETE /movies/{id}: Remove a movie from the database.

Frequently Asked Questions (FAQ)

How do I reset my password?

You can reset your password by clicking on the "Forgot Password" link on the login page and following the instructions.

Can I import movies from other databases?

Yes, MovieX supports importing movies from various formats including CSV and JSON.
